How I setup my own personal cloud


Hardware:

  • CPU: AMD Phenom II X6 1055T (overclocked to 4GHZ)
  • Motherboard: ASUS Crosshair V Formulala>
  • GPU: AMD Radeon 7770
  • NIC: Mellanox ConnectX-2 10Gbps SFP+
  • Storage: Samsung 840 500GB Evo SSD boot drive
  • WD Black 8TB + numerous old HDDs
  • Total Raw Storage = 15TB

Operating System(s):
  • Proxmox VE type 1 hypervisor (the only available free and open source bare-metal hypervisor)
  • Ubuntu Server 20.04
  • Windows 10 (with GPU Passthrough configured for gaming)

Storage Configuration:
  • 2x2TB ZFS Stripe
  • 3x1TB ZFS Stripe
  • 1x8TB for backup

Here's a list of all the services self hosted on the Server:
  • Dashboard to access all the different services: Heimdallhome.kvudit.com
  • File Storage: Nextcloudstorage.kvudit.com
  • Media Streaming: Streama (Your own private Netflix!)watch.kvudit.com
  • Media Streaming(Including a music player): Embymedia.kvudit.com
  • WebServer: ApacheThis Website
  • Monitoring: Grafanamonitor.kvudit.com
  • IoT: Home Assistant installed in a LXCcontrol.kvudit.com
  • Proxy: nginx serving as a reverse proxy for all servicesproxy.kvudit.com
  • Web Analytics: Ackeeanalytics.kvudit.com
  • Surveillance: MotionEyecamera.kvudit.com
  • Server Management: Cockpit cockpit.kvudit.com
  • VPN: OpenVPN
