Setting up your own personal cloud

So I decided to repurpose my gaming PC from 10 years ago and make it my own personal server.
Hardware:
CPU: AMD Phenom II X6 1055T (overclocked to 4GHZ)
Motherboard: ASUS Crosshair V Formula
GPU: AMD Radeon 7770
NIC: Mellanox ConnectX-2 10Gbps SFP+
Storage: Samsung 840 EVO 500GB SSD boot drive
WD Black 8TB + numerous old HDDs
Total Raw Storage = 15TB
Operating System(s):
Proxmox VE type 1 hypervisor (the only available free and open source bare-metal hypervisor)
Ubuntu Server 20.04
Windows 10 (with GPU Passthrough configured for gaming)
Storage Configuration:
2x2TB ZFS Stripe
3x1TB ZFS Stripe
1x8TB for backup
Here's a list of all the services hosted on the Server:
File Storage: Nextcloud
Media Streaming: Streama (Your own private Netflix!)
VPN: OpenVPN
WebServer: Apache
Monitoring: Grafana
IoT: Home Assistant installed in a LXC
Proxy: nginx serving as a reverse proxy for all services
